CSS模糊效果实践

2016年12月1日CSS

初始状态

<!-- html -->
<div class="bgimg">
  <div class="blur">模糊效果测试</div>
</div>

<!-- css -->
.bgimg{
    height: 500px;
    background-image: url(bgpic6.jpg);
    background-size: contain;
    background-repeat: no-repeat;
    overflow: hidden;
}
.blur {
    width: 200px;
    height: 200px;
    margin: 150px 250px;
    background-color: hsla(0,0%,100%,.4);
}

这里使用hsla颜色值加透明度显示一个文字层的效果，但是并没有达到模糊的效果

加入遮罩层

需要在文字层下方新建一个模糊层遮罩，可以在.blur元素里添加一个子元素，也可以直接添加一个伪元素

.blur {
    width: 200px;
    height: 200px;
    margin: 150px 250px;
    positon: relative;
    z-index: 0;
}

.blur:after {
    content: '';
    position: absolute;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
    z-index: -1;
    background-color: yellow;
}

如果在.blur:after上不设置z-index，after伪元素会遮住文字

z-index为负值时，会将该元素设置在第一个有定位(position非static)且设置了z-index值得父元素上，这里我们设置.blur元素为position: relative; z-index: 0，所以.blur:after伪元素会位于.blur元素上，text文字下，且会遮盖设置在.blur元素上的任何背景

如果换在.bgimg元素上设置，则.blur:after伪元素位于.bgimg元素上，.blur元素下，这时如果我们给.blur元素设置一个背景色，会发现.blur:after伪元素被完全遮盖了

.bgimg {
    positon: relative;
    z-index: 0;
}

.blur {
    /* positon: relative; */
    /* z-index: 0; */
    background-color: wheat;
}

遮罩层添加背景图

使用background-attachment属性的fixed值，使背景图层和遮罩层的图片完全重合。background-attachment用来设置背景图的位置是固定在视图窗口上，还是跟随元素一起滚动

这里设置了一个border值来突出.blur元素

.bgimg{
    height: 500px;
    background-image: url(bgpic6.jpg);
    background-size: contain;
    background-repeat: no-repeat;
    background-attachment: fixed;
    overflow: hidden;
}
.blur:after {
    content: '';
    position: absolute;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
    z-index: -1;
    background-image: url(bgpic6.jpg);
    background-size: contain;
    background-repeat: no-repeat;
    background-attachment: fixed;
}

使用filter: blur()实现模糊效果

filter: blur(5px)指定一个模糊半径，可以非常方便地实现模糊效果。但只能在高级浏览器下使用，IE不兼容

该模糊效果会在边缘弱化，所以设置遮罩层的margin为负值使其size大于.blur元素，再在.blur元素上设置overflow: hidden切掉超出部分，可得到一个理想的模糊效果

.blur {
    overflow: hidden;
}
.blur:after {
    filter: blur(5px);
    margin: -10px;
}

完整代码

<!-- html -->
<div class="bgimg">
  <div class="blur">模糊效果测试</div>
</div>

<!-- css -->
.bgimg{
    height: 500px;
    background-image: url(bgpic6.jpg);
    background-size: contain;
    background-repeat: no-repeat;
    background-attachment: fixed;
    overflow: hidden;
}
.blur {
    width: 200px;
    height: 200px;
    margin: 150px 250px;
    position: relative;
    z-index: 2;
    overflow: hidden;
}
.blur:after {
    content: '';
    position: absolute;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
    z-index: -1;
    background-image: url(bgpic6.jpg);
    background-size: contain;
    background-repeat: no-repeat;
    background-attachment: fixed;
    filter: blur(5px);
    margin: -10px;
}

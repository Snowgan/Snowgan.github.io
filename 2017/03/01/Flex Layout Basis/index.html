Flex Layout Basis

2017年3月1日CSSflex

We can create a flexbox container by setting the display property to flex or inline-flex, and the direct children of the container will become flex item.

Flexbox container have two axes: the main axis and the cross axis which indicate how the flex items place.

flex-direction define the main axis direction, have four values to choose: row(defualt), row-reverse, column, column-reverse. The cross axis runs perpendicular to the main axis. The flex items will be placed according to the main axis direction.

By default,

  • The flex items do not stretch but can shrink on the main axis direction
  • The flex items will stretch to fill the size of the cross axis.
